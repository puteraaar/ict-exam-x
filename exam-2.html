Case Study: Student Permission Form

Context

To streamline administrative tasks and maintain accurate records, the school needs a digital system for students requesting permission to leave the campus during school hours. Your task is to develop a clear and formal permission slip form for the school's official website.

Task

Create a web page that features a permission slip form for students needing to leave school grounds. The form must include the following elements, styled appropriately using Tailwind CSS:

  1. VStudent's Full Name: A text input field for the student's full name.
  2. Class: A selection field with options for different classes, such as "X.1", "X.2", "X.3".
  3. Date of Leave: A date input field for the day the student needs to leave.
  4. Departure Time: A time input field for when the student plans to leave the school.
  5. Estimated Return Time: A time input field for when the student expects to be back.
  6. Type of Leave: A group of radio buttons with the options: Sick Leave and Other Appointment.
  7. Brief Reason: A text input field with a placeholder text like "Dentist Appointment, Family Matter, etc.".
  8. Approving Teacher's Name: A text input field for the name of the teacher who gave initial permission.
  9. Detailed Description: A textarea for a more detailed explanation of the reason for leaving, if necessary.
  10. Submit Button: A button labeled "Submit Request".

Challenge Plus Point

Add a hover effect to the "Submit Request" button, so its color changes when the cursor hovers over it.

Submission

  1. Save your file as name_class_psts.html. (eg. Putera Islamiyadi_X1_psts.html)
  2. Ensure the code is well-structured and easy to read.
