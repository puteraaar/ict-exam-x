Practical Exam Outline

Web Programming using HTML and Tailwind CSS

Device Used

Device: Laptop

Code Editor: Visual Studio Code

Exam Outline

  1. You should be able to correctly write the basic structure of an HTML document, including the <!DOCTYPE html>, <html>, <head>, and <body> tags.
  2. You'll be tested on your ability to use essential HTML form elements:
    • <form> as a container for all form elements.
    • <input>for various data types (text, email, password, etc.).
    • <select> and <option>for creating a dropdown menu.
    • <textarea>for accepting long, multi-line text.
  3. You should be able to know how to use key attributes within form elements. This includes:
    • type attribute (e.g., text, email, date, time, radio, checkbox)
    • placeholderto provide a hint or example for the input field.
  4. You need to be able to implement various Tailwind CSS utility classes to style your web page. Key classes include:
    • Layout: flex, grid, flex-col, grid-cols-* for arranging elements.
    • Sizing: w-full, p-*, m-* to control width, padding, and margin.
    • Color: bg-*, text-* for setting background and text colors.
    • Visual: rounded-*, shadow-* to create rounded corners and shadows.
  5. You should be able to use pseudo-classes like hover: to create interactive effects. For example, using hover:bg-blue-700 to change a button's background color when a user's cursor hovers over it.
